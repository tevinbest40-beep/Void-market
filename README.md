# VOID MARKET ⚡

A cyberpunk-style marketplace for buying and selling eFootball accounts.

## 🚀 Features
- Post eFootball accounts instantly
- Add price, image, and contact
- WhatsApp direct buy button
- Cyberpunk UI design
- Lightweight and fast (no backend)

## 🎮 Built For
eFootball players who want a simple trading platform.

## 🌐 Live Site
Hosted on GitHub Pages

## ⚙️ Tech Stack
- HTML
- CSS (Cyberpunk theme)
- JavaScript

## 📌 Notes
- No login system
- No database (posts are temporary)
- Images use external links

## 💡 Future Upgrades
- Firebase database
- Real image uploads
- Search & filters
- User profiles
